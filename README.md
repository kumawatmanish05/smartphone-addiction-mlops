# 📱 Smartphone Addiction Prediction (MLOps Project)

🚀 An end-to-end Machine Learning project that predicts a user's smartphone addiction level based on behavioral usage patterns.

---

## 🔥 Features

- 📊 Data Cleaning & Feature Engineering Pipeline  
- 🤖 Machine Learning Model (Multi-class Classification)  
- ⚙️ Modular Code Structure (`src/`)  
- 🌐 FastAPI for Real-time Predictions.
- 📦 Model & Encoder Persistence using `joblib`  
- 📈 Production-ready ML workflow  

---

## 🧠 Problem Statement

With increasing smartphone usage, identifying addiction levels is crucial.

This project predicts:
- **Mild**
- **Moderate**
- **Severe**

based on user behavior such as:
- Screen time  
- Social media usage  
- Sleep patterns  
- Notifications & app activity
